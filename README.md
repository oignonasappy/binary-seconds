- https://oignonasappy.github.io/binary-seconds/
  This represents the time until a 32-bit integer Unix time is exhausted.
- https://oignonasappy.github.io/binary-seconds/?64bit  
  This represents the time until a 64-bit integer Unix time is exhausted.
- https://oignonasappy.github.io/binary-seconds/?ms  
  This represents the time until a 64-bit integer Unix time in milliseconds is exhausted.
> [!NOTE]
> `?ms` appears to be slow in the first 4 bits, but this is due to the nature of `Date.now()`.
